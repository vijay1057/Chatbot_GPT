6. Refresh the data
When using the full sync option in BI sync, it is possible to use the same Power BI configuration and load new data when it is
available. You must use the same directory and naming convention used when the Power Bi configuration and import was
initially performed. After the new data is copied into the folder the data was loaded into, import it into Power BI by right
clicking on each of the csv files and clicking ‘refresh’ to get the updates.
From inside Power BI, click on every file and select refresh.
