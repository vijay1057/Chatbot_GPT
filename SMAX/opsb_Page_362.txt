UCMDB Integration
Introduction
Service Management Automation X provides an integration with the UCMDB that synchronizes Service Asset & Configuration
Management (SACM) information. The UCMDB integration uses the On-Premises Bridge (OPB) to connect to an external UCMDB
to synchronize Actual Services, Service Components, System Elements and Assets to the Service Management Automation X
repository. This document provides detailed information regarding the configuration of this integration.
This information includes:
Installing the OPB Agent
Creating an Endpoint
Configuring SSL for OPB
Configuring the UCMDB Endpoint
Verifying Data Synchronization
