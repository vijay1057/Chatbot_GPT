Once the Dashboard has opened, go to the Large Menu and Select 
Administration
 -> 
Utilities
 -> 
Integration
.
Note: This assumes all previous steps to install OPB Agent have already been completed on an appropriate server.
Click on 
Endpoints
.
Within the Endpoints screen click on 
Add
. Here a new Windows will appear asking for
 Endpoint type
, 
Endpoint name
 and
Running on agent
. Fill in the details like so and click 
Add
.
This has now created an Endpoint for OO on the OPB.
Configure the OO Endpoint

Endpoint that was created earlier needs to be configured. Go to the Endpoint and click on 
Configure.
