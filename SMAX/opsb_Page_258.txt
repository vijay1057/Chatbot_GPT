Within the Details screen for the Product, click on 
License Rule Settings
.
Click on the Product Version, in this case 
McAfee VirusScan 8 Ent
.
At this point the Default rule must be assigned to the Product (or no consumption will be taken into account). Once the Default
Rule has been applied, SAM associates discovered products to the License Model and License information that was created
earlier.
When the License Model was created it was based upon the Generic per Device rule. So for this product, that check box needs
to be ticked and then the Default Value will automatically be set to that rule.
