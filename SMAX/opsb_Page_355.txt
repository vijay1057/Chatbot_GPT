Run through the Password Reset Scenario
A user of the eExpenseDB can no longer access the database and needs to get their password reset.
Log in to SMAX with the user who would like to reset their password for the eExpenseDB system.
From the main portal page, click on Business Applications.
Click on Request Support for the Password reset offering for eExpense.
Fill in the form describing the problem and then click on 
Submit
. The key values have already been populated based on what
was entered for the Default Values within the Offering.
