Licensing
The solution can be deployed with the following licenses: 
SMAX Express and SMAX Premium: provide the access to IT Service Management and Extended Service Management
solutions.
AMX: provides the access to a consolidated Asset Management solution. A SMAX Premium license is required if you would
like to include Asset Management capability in SMAX. AMX can also be deployed independently if you intend to use Asset
Management capabilities only.
HCMX: provides the access to a consolidated Cloud Management solution. An HCMX license is required if you would like to
include Cloud Management capability in SMAX, it can also be deployed independently if you intend to use Cloud
Management capabilities only.
