Multi-Tenant and Data domain rules 
With Native SACM, the tenant model and data domain rules in SMAX are fully transparent to UCMDB. In one farm, every SMAX
tenant can have its own UCMDB customer to be uniquely connected with in the same UCMDB instance. You no longer need to
set up another UCMDB instance for each SMAX tenant. This also applies to the logical data separation called Data Domain
Segmentation in SMAX. This feature works transparently with UCMDB to ensure data security across the whole suite.
